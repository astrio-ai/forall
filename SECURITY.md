# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Forall or this repository, please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

Email security reports to the maintainers at Astrio AI. Include:

- A description of the issue
- Steps to reproduce
- Impact assessment, if known

We will acknowledge receipt and work on a fix. We ask that you allow reasonable time for remediation before public disclosure.

## Scope

This policy covers:

- The `install.sh` script in this repository
- Documentation and community assets published here
- Prebuilt `forall` binaries distributed via [GitHub Releases](https://github.com/astrio-ai/forall/releases)

Agent source code is not published in this repository.
